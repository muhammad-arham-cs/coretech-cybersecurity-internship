Cybersecurity Q&A

What is Cybersecurity?
It's basically the practice of keeping networks, devices, and data safe from unauthorized access or malicious attacks. Think of it as the digital defense system that keeps a company's assets and user info secure.

What are the main types of cyber threats?
Malware: Bad software like viruses or ransomware that messes up your system or locks your files.
Phishing: Social engineering tricks (like fake emails) used to steal passwords or sensitive info.
DoS/DDoS: Flooding a server with so much junk traffic that it crashes and real users can't access it.
Man-in-the-Middle (MitM): When an attacker secretly intercepts the communication between two parties to steal or alter data.

What is the CIA Triad?
Confidentiality: Making sure only the right, authorized people can see the data.
Integrity: Making sure the data hasn't been tampered with or changed secretly.
Availability: Making sure the systems and data are actually up and running when authorized users need them.

What is ethical hacking?
It's legally breaking into systems to find the weak spots before the bad guys do. Ethical hackers use the exact same tools as malicious hackers, but their goal is to report and patch the vulnerabilities, not exploit them.

How can cybersecurity help CoreTech Innovation?
Good cybersecurity means CoreTech can build and launch its projects without constantly stressing about data breaches or stolen intellectual property. It protects client data, prevents expensive downtime, and ultimately proves to customers that CoreTech is a reliable company they can trust.
